# キーボードショートカット一覧

> macOS では Ctrl の代わりに Cmd（⌘）を使用

## ファイル操作

| ショートカット | 動作                   |
| -------------- | ---------------------- |
| Ctrl+N         | 新規タブを作成         |
| Ctrl+O         | ファイルを開く         |
| Ctrl+S         | 上書き保存             |
| Ctrl+Shift+S   | 名前を付けて保存       |
| Ctrl+W         | アクティブタブを閉じる |

## 編集

| ショートカット | 動作                  |
| -------------- | --------------------- |
| Ctrl+E         | 閲覧 / 編集モード切替 |
| Ctrl+Z         | 元に戻す（Undo）      |
| Ctrl+Shift+Z   | やり直し（Redo）      |
| Ctrl+F         | 検索バーにフォーカス  |

## タブ操作

| ショートカット              | 動作           |
| --------------------------- | -------------- |
| Ctrl+PageDown, Ctrl+Tab     | 次のタブへ切替 |
| Ctrl+PageUp, Ctrl+Shift+Tab | 前のタブへ切替 |
